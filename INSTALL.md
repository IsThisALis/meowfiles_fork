## What and how to install
### Dependencies

1. SwayFX
2. Wofi
3. Wezterm
4. Waybar
5. Fish - agave-nerd font

**Optional**
1. NeoVim
2. BongoCat

### Install

**Arch**:

```bash
sudo pacman -S wofi wezterm waybar ttf-agave-nerd
```

SwayFX (yay):

```bash
yay swayfx
```

SwayFX (paru):

```bash
paru swayfx
```

Optional:

BongoCat (yay):

```bash
yay bongocat
```

BongoCat (paru):
```bash
paru bongocat
```

NeoVim:

```bash
sudo pacman -S neovim rust
```
